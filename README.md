# Landing lista para Netlify

## Deploy

- Opción 1: arrastrar y soltar la carpeta del proyecto en Netlify (Deploys → Manual deploy).
- Opción 2: conectar el repo (GitHub/GitLab/Bitbucket).

### Configuración

- `netlify.toml` define `publish = "."` (sitio estático, sin build).
- El formulario usa **Netlify Forms** (ver sección Contacto en `index.html`).

## Personalización rápida

- Reemplazá `pedrazzolifacundo@gmail.com`, el link de WhatsApp (`https://wa.me/...`) y el link de agenda.
- Cambiá el nombre `Mi Propuesta` en el header/footer.
